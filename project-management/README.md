# project-management

Express app for project management using Node.js, Sequelize, and PostgreSQL.

## Local development

### Start postgres

Refer [README.md](/apps/backend/README.md)

```shell
yarn nx run backend:start:local
```

### Sequelize setup (If not done already)

Refer [README.md](/apps/project-management/DOCS/SEQUELIZE-SETUP.md)

### Start the development server

```shell
yarn nx run project-management:serve
```
